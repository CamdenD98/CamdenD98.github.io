Finding Factors

Here I have some Java code where the user would input two numbers and the program would determine if they are factors of one another. 


                import java.util.Scanner;
                public class Factors {

	                public static void main(String[] args) {

		                System.out.println("Enter first number: ");
		                int userFirst = input.nextInt();

		                int userSecond = input.nextInt();

		                else if (userSecond%userFirst ==0)

Input Your Grade for a Message

Here I have some Java code where the user would input their current grade and the program would output a message based on the grade. 


                import java.util.Scanner;
                public class Grades {

	                public static void main(String[] args) {
		            Scanner input = new Scanner(System.in);
		            System.out.println("Enter your current grade: ");
		            String userGrade = input.next();
		            String lower;
		            lower = userGrade.toUpperCase();
		
		            switch (lower) 
		            {
		            case "A" : System.out.println("Good Job");break;
		            case "B" : System.out.println("Doing well");break;
		            case "C" : System.out.println("Work a little harder");break;
		            case "D" : System.out.println("STEM Ambassadors lead peer-led study sessions every Wednesday evening in GH 230 from 7-9pm.");break;
		            case "F" : System.out.println("Danger, Will Robinson");break;
		            default : System.out.println("You lost me."); break;
		            }
	            }

            }
